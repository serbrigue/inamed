Introducción:

Esta es una responsive web diseñada por alumnos de primer año de la carrera Ingenieria en Informática INACAP Valparaíso, el cual ofrece una vista más realista del diseño presentado para la asignatura Modelamientos de Soluciones Informáticas con el docente Oscar Ascencio Alarcón.

Presentación:

La presentación de esta web, contempla varios pasos de elaboración.

1.- Concretar la idea de elaboración de la web, integrando una página inicial, gestión de usuarios, a través de un login para clientes / trabajadores (médicos), pacientes para realizar citas médicas, médicos para consultar citas agendadas, historial de paciente, actualizar tratamientos, realizar observaciones, cancelar citas.

2.- Diseño de flujograma para la elaboración y asignación de trabajo, mockup con el prediseño, feedback de mejora.

3.- Elaborar el diseño en distintos stacks a utilizar, los cuales se definieron Javascript, Tailwind, React.
